Introduction

ToolDAQ.js is a JavaScript library for interacting with the back-end services for monitoring and data systems, providing functionalities for database queries, requests and data visualization. ToolDAQ.js provides functionalities for:

  • Database Queries: Retrieve and manage structured data from the ToolDAQ databases efficiently.
  • Requests: Send and handle requests to interact with external services and endpoints.
  • Data Visualization: Create plots and visualizations for data analysis using plotly.
  • Service Management: Interact with and manage ToolDAQ services for seamless integration.
  • Utilities: Miscellaneous helper functions to enhance development workflows.

Examples of some of the functionalities are shown with interactive demonstrations on the following pages:

  • Database Query Operations
  • Plotting and Visualization Functions
  • Request Functions
  • Service Functions

Page Creation

To create a page called myPage, copy any of the existing directories in html-Detector. This should contain the following files:

  • myPage/index.html
  • myPage/subheader.html
  • myPage/tablist
Sub-pages can be stored in the myPage folder. This should just have an index page:
  • myPage/mySubPage/index.html
The page index.html should have the following format: 
<!DOCTYPE html>
<html lang="en">
<head>
  <title>Title</title>
</head>
<body>
  <!--#include virtual="/includes/header.html" -->
  <!--#include virtual="subheader.html" -->
  <!--#include virtual="/includes/drawer.html" -->
            
  Content here
            
  <!--#include virtual="/includes/footer.html" -->
</body>
</html>

Importing ToolDAQ.js

To use ToolDAQ.js in your project, follow these steps:

  1. Create a new JavaScript file (e.g. app.js) where you'll write your code.
  2. Import the functions you need from tooldaq.js by using the ES6 import syntax: 
    import { dbJson } from '/includes/tooldaq.js';
    If you need multiple functions, you can import them simultaneously: 
    import { dbJson, dbTable, dbPlot } from '/includes/tooldaq.js';
  3. Link your JavaScript file to your HTML page by adding the following line to your HTML file to include your app.js: 
    <!DOCTYPE html>
<html lang="en">
<head>
  <title>ToolDAQ.js WebPage</title>
</head>
<body>
  <!--#include virtual="/includes/header.html" -->
  <!--#include virtual="../subheader.html" -->
  <!--#include virtual="/includes/drawer.html" -->

  <h1>ToolDAQ.js WebPage</h1>

  <!--#include virtual="/includes/footer.html" -->
  <script src="app.js"></script>
</body>
</html>

Once you've completed these steps, you're ready to start using ToolDAQ.js!
